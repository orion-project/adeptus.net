# Adeptus

Adeptus is a simplistic issue, task, and idea-tracking application for personal projects.

This variant is a rewrite of the original [Qt version](https://github.com/orion-project/adeptus) with .NET and the [Avalonia](https://github.com/AvaloniaUI/Avalonia) cross-platform UI framework used mainly as a playground for learning Avalonia.
